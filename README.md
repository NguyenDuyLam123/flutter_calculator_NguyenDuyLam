# Simple Mobile Calculator - Flutter (LAB 2)

## Description
This is a simple mobile calculator app built using Flutter. The UI is based on the Figma design from the lab instructions.

## Features
- Basic operations: +, -, ×, ÷
- Percentage (%)
- Sign toggle (±)
- Decimal input
- C (clear all)
- CE (clear last digit)
- Error handling (divide by zero)
- Long number scrolling
- UI matches Figma (colors, spacing, border radius)
